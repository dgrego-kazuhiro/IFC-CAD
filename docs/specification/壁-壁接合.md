6. Wall ↔ Wall 接合アルゴリズム
6.1 入力
interface WallElement {
  id: string;
  axis: [Vec3, Vec3];
  thickness: number;
  height: number;
  joinStart: boolean;
  joinEnd: boolean;
}
6.2 接合条件
endpoint 同士が tolerance 内
endpoint が他壁 axis 上に乗る
2軸線が交差する
6.3 Butt Join
意味

片方の壁が他方の壁面で止まる。

手順
壁Aと壁Bの軸線交点 P を求める
壁Bの左右 face line を計算
壁A の接合端から見て前進方向にある B face との交点を採用
A の端点をその位置まで延長/短縮
B は通常そのまま
出力
wall A endpoint override
wall A end cap plane override
6.4 Miter Join
意味

両壁の厚み境界を突き合わせて斜めに納める。

手順
A/B の軸線交点 P
各 wall の左右 offset line を作る
対応する外側/内側線の交点を計算
4角形または端部輪郭を再構築
両壁の端面を斜めに定義
6.5 T字接合
手順
wall A endpoint が wall B の axis 上または厚み内にある
B の face plane を求める
A endpoint を B face まで trim
B は分割しない（MVP）
将来、必要なら B surface に接合痕 metadata を持つ
6.6 Wall-Wall 出力データ
interface WallJoinResult {
  wallId: string;
  at: "Start" | "End";
  trimPoint?: Vec3;
  trimPlane?: Plane;
  rebuiltFootprint?: Vec2[];
}
