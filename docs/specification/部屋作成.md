Hyparの壁厚に関する仕様まとめ
1. 基本仕様とデフォルト設定
自動生成: スペースを作成したりプログラムスペースをインポートしたりすると、壁が自動的に追加されます。
標準の厚さ: デフォルトの壁厚は、**5.25インチ（133mm）**に設定されています。
面積への影響: 壁は常に外側に向かって成長（厚くなる）スペースの面積には影響しません。

2. 編集と操作
個別の調整: 壁を選択してドラッグするか、数値を入力することで厚さを変更できます。厚さを**「0」に設定**して壁をなくすことも可能です。
セグメント化: 「Edit Shape」でポイントを追加することで、壁のセグメントごとに異なる厚さを持たせることができます。
一括編集: Shiftキーを押しながらの選択や、ボックス選択ツールを使用して、複数の壁を同時に編集できます。
削除と復元: カプセルメニューの壁ボタンから、壁の全削除や再配置が簡単に行えます。

3. 挙動と精度
自動結合: 隣接する壁は、厚さに関わらず自動的に結合され、壁の重複（オーバーラップ）は発生しません。
スナップ機能: ピンク色のスナップガイドとインジケータにより、素早い操作でも精度が維持されます。
カスタムの記憶: Hyparはユーザーが設定したカスタムの壁厚を記憶し、作業効率を向上させます。

4. 視覚的仕様（3D表示）
動的な非表示: 内部を見やすくするため、手前にある壁は自動的に非表示になります。
選択時の挙動: スペースを選択している間、家具などのアイテムを選択しやすくするために、すべての壁が一時的に非表示になります。

5. Revitとの連携
アドインによる変換: 専用のアドインを通じて、モデルをRevitに変換できます
。
クリーンな移行: 移行時、Hyparは重複した壁を削除し、クリーンに結合された「Generic Hypar Walls」としてRevit上に生成します
。
