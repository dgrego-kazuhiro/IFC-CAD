# タスク: CADアプリケーション状態管理と壁作成機能の実装

## 目的
前回の続きとして、WebGL/WebGPUを用いたIFC指向CADプロトタイプの基盤となるアプリケーションステート（Zustandを使用）の構築と、それを利用した「壁(Wall)」要素の作成および描画機能を実装します。

## 対象範囲
1. **AppState / AppContextの構築** (Zustandを用いたステート管理)
2. **コマンド層との統合** (`CreateWallCommand` など)
3. **UI層の更新** (`CadShell.tsx`, `Viewport.tsx` をステートに連動させる)
4. **ジオメトリ/メッシュ生成** (壁の基本形状およびメッシュ変換関数の実装)
5. **レンダラーの更新** (シーン内の要素を描画)

## 未対象範囲 (MVP後)
- 複雑な壁同士の結合（現在は単純な押し出しのみ）
- IFCへの完全なエクスポート機能
- スペース（空間）検出機能
