# Wall Sketch 機能追加タスクリスト

- [x] WallSketchState: 状態の型定義とオプション設定の作成
- [x] WallSketchPreview: プレビュー時の表示用セグメント生成ヘルパー関数の作成
- [x] WallSketchController: イベントハンドリングおよび状態遷移のメインロジックの構築
- [x] WallTool: Controllerへの入力イベントを中継するツールの統合
- [x] SnapManager: 仮スナップ判定ロジックの実装（グリッドスナップ等）
- [x] WallJoinResolver: 簡易的な交差/接続のスタブ実装
- [x] TransactionCommand: 一連の連続入力（Polylineモード等）を一つのコマンド履歴にまとめるための複合コマンドの実装
